# 🎨 UI Компонент: CompactColorPicker

## 1. Назначение и Облик
Компактный контроллер для выбора цвета, объединяющий визуальное превью выбранного цвета (или радужный градиент если не выбран), HEX-значение и кнопку пипетки (через EyeDropper API).

Визуальные особенности:
- **Indicator**: Градиентное превью "радуга", если цвет не выбран.
- **Typography**: Использование `font-black` и `tabular-nums` для HEX-кода.
- **Micro-interactions**: Иконка пипетки (`Pipette`), при нажатии активируется браузерная пипетка.
- **Popover**: При клике на саму плашку выплывает полноценный `ColorPicker`.

## 2. Пропсы (Props)
```typescript
interface ColorPickerCompactProps {
  value?: string;           // Текущее значение цвета
  onChange?: (color: string) => void; // Колбэк при выборе цвета
}
```

## 3. Состояния (States)
- **Closed:** Отображается компактная плашка с HEX-кодом.
- **Open:** Поверх плашки выплывает `ColorPicker`.
- **Empty:** Если `value` отсутствует, отображается надпись "Выбрать" и радужный индикатор.

## 4. Особенности реализации
- **EyeDropper API**: Нативная пипетка браузера для выбора цвета с экрана (работает только в поддерживаемых браузерах, Chromium-based).
- **Синхронизация**: Принимает `value` и правильно синхронизирует его с внутренним состоянием `hexColor`.
