# 🎨 UI Компонент: RoleColorPicker

## 1. Назначение и Облик
Специализированный компонент для выбора цвета роли или статуса из предустановленного набора. Используется в административной панели для настройки цветовой идентификации пользователей.

Визуальные особенности:
- **Grid Layout**: Элементы расположены в гибкой сетке с отступами.
- **Glass-Inner**: Контейнер имеет фон `slate-50` и внутреннюю тень для создания эффекта "вдавленности".
- **Peer-Checked**: Использует скрытые радио-кнопки и CSS-состояния `peer` для визуализации выбора.

## 2. Пропсы (Props)
```typescript
interface RoleColorPickerProps {
    defaultValue?: string; // Значение цвета по умолчанию (из списка ROLE_COLORS)
}
```

## 3. Доступные цвета (`ROLE_COLORS`)
Компонент поддерживает следующий набор токенов:
- `indigo`, `slate`, `red`, `orange`, `emerald`, `blue`, `purple`, `rose`, `cyan`, `amber`.

## 4. Особенности реализации
- **No-JS Select**: Выбор работает на основе стандартных HTML-радиокнопок, что обеспечивает базовую доступность и надежность.
- **Visual Feedback**: При выборе элемент получает белое кольцо и внешнюю обводку цвета самого элемента.
