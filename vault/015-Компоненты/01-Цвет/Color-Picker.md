# 🎨 UI Компонент: ColorPicker

## 1. Назначение и Облик
Полнофункциональный инструмент выбора цвета в эстетике **Lumin-Apple**. Поддерживает плавный выбор через HSV-поле, слайдер оттенка (Hue), автоматическую генерацию случайного цвета и прямой ввод HEX-кода.

Визуальные особенности:
- **Rounded-20**: Радиус скругления контейнера 20px.
- **Glassmorphism**: Легкая граница и глубокая мягкая тень.
- **Micro-animations**: Плавные переходы при наведении на пресеты и кнопки.

## 2. Пропсы (Props)
```typescript
interface ColorPickerProps {
  color?: string;       // Текущий цвет (HEX, по умолчанию #73C242)
  onChange?: (color: string) => void; // Колбэк при изменении
  className?: string;   // Дополнительные стили контейнера
}
```

## 3. Состояния (States)
- **Default:** Отображает палитру пресетов (9 цветов), HSV-область, Hue-слайдер и поле ввода.
- **Randomize:** При клике на кнопку Shuffle генерируется случайный HSV-цвет.
- **Input Sync:** При ручном вводе HEX-кода в инпут, позиция курсора на палитре обновляется автоматически.

## 4. Особенности реализации
- **HSV Logic**: Внутреннее состояние хранится в HSV для точности преобразований, но экспортируется как HEX.
- **Pointer Capture**: Использует `setPointerCapture` для стабильного перемещения курсора даже за пределами области компонента.
