# 🎨 UI Компонент: ColorGroup

## 1. Назначение и Облик
Компонент для быстрого выбора цвета из фиксированного набора образцов (палитры). Используется там, где неоправданно открытие полного пикера, например, при категоризации мелких сущностей.

Визуальные особенности:
- **Radio-style**: Каждый образец ведет себя как радио-кнопка с визуальной индикацией выбора (галочка `Check`).
- **Focus Rings**: Четкие кольца фокусировки для навигации с клавиатуры.
- **Micro-interactions**: Увеличение масштаба (`scale-110`) при наведении.

## 2. Пропсы (Props)
```typescript
interface ColorGroupProps {
  selectedColor?: string;              // Выбранный HEX-цвет
  onColorChange?: (color: string) => void; // Колбэк при клике
  colors?: string[];                   // Массив HEX-строк (опционально)
  className?: string;                  // Стили контейнера
}
```

## 3. Стандартная палитра (`_SWATCH_COLORS`)
Если пропс `colors` не передан, используется системный набор MerchCRM:
- Rose, Fuchsia, Violet, Blue, Cyan, Emerald, Lime.

## 4. Особенности реализации
- **Accessibility**: Имеет роль `radiogroup` и соответствующие `aria-label` для скринридеров.
- **Indicator**: Выбранный цвет выделяется не только галочкой, но и внутренним отступом (`boxShadow: inset...`), что создает эффект приподнятости.
