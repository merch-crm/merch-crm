# 🎨 UI Компонент: ColorSwatch (ColorPickerSquare & ColorPickerCircle)

## 1. Назначение и Облик
Минималистичные элементы выбора цвета, которые выглядят как квадрат или круг. Идеально подходят для случаев, когда место ограничено, а выбор цвета второстепенен.

При клике на Swatch открывается Radix `Popover` с полноценным `ColorPicker`.

Визуальные особенности:
- **Формы**: Квадрат со скругленными углами (`ColorPickerSquare`) или круг (`ColorPickerCircle`).
- **Стиль "Пустоты"**: Если цвет не выбран, отображается радужный градиент (`RAINBOW_GRADIENT`).
- **Тень и Границы**: Легкая обводка `border-black/5` для цвета, чтобы он не сливался со светлым фоном.

## 2. Пропсы (Props)
Оба компонента принимают одинаковый интерфейс:

```typescript
interface ColorPickerTriggerProps {
  value?: string;           // Текущий выбранный цвет
  onChange?: (color: string) => void; // Вызывается при выборе
  label?: string;           // Метка, показываемая рядом или над
  className?: string;       // Дополнительные стили контейнера
}
```

## 3. Особенности реализации
- **Встроенный Popover**: Компоненты полностью инкапсулируют логику попапа (`useReducer` для isOpen).
- **Fallback**: Если компонент не контролируемый (без `value`/`onChange`), он умеет управлять своим внутренним состоянием самостоятельно.
- **Внешний вид**: Активно используют Tailwind `ring`, `focus-visible` и `active:scale-95` для микроинтеракций при фокусе и нажатии.
