# 02. Кнопки

Система кнопок MerchCRM основана на 6 семантических цветах и 4 визуальных стилях. Это основной интерактивный элемент системы.

## Навигация

- [[Button]] — базовый компонент.
- [[SubmitButton]] — кнопка для отправки форм с состоянием загрузки.

## Цветовая палитра

| Цвет | Назначение | HEX (Solid) |
|---|---|---|
| `dark` | Основное действие | `#020617` (Slate 950) |
| `danger` | Опасные / Деструктивные | `#E11D48` (Rose 600) |
| `warning` | Предупреждения | `#F59E0B` (Amber 500) |
| `system` | Вспомогательные / Брендовые | `#4F46E5` (Indigo 600) |
| `success` | Успех / Позитивные | `#10B981` (Emerald 600) |
| `neutral` | Вторичные / Системные | `#F1F5F9` (Slate 100) |

## Стили (Variants)

1. **Solid**: Основной стиль с заливкой.
2. **Outline**: Стиль с контуром для вторичных действий того же цвета.
3. **Ghost**: Минималистичный стиль без фона (фон появляется при наведении).
4. **Link**: Текстовая ссылка (только для цвета `neutral`).

## Влияние на дизайн

Все кнопки следуют стандарту **Modern Industrial Craft**:
- Закругление: `2xl` (16px) для размеров `md`, `lg`, `icon`. `xl` (12px) для `sm`.
- Высота: Фиксированная (`46px` для `md`).
- Шрифт: Bold, Tracking Normal.
