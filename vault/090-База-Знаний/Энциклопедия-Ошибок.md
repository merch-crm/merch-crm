# 📕 Энциклопедия Ошибок
База знаний по типовым ошибкам в MerchCRM, методам их диагностики и исправления.

## 🚨 Реестр инцидентов (Post-mortem)

| Дата | Ошибка | Причина | Решение |
| :--- | :--- | :--- | :--- |
| 2026-04-03 | `read ECONNRESET` при подключении к БД через SSH-туннель | Туннель воркфлоу `/start_dev_session` указывал на `127.0.0.1:5432` хоста, но Postgres в Docker **не публикует порт** (нет `ports:` в docker-compose). | Исправлен на использование `node scripts/setup-ssh-tunnel.mjs`, который динамически получает IP контейнера (`172.18.0.4`) и пробрасывает порт напрямую к нему. |
| 2026-04-02 | `500 Internal Server Error (Database auth failed)` | Сброс пароля `postgres` через `ALTER USER` в workflow `start_dev_session`. | Восстановлен оригинальный пароль из `.env` прода. Локальный `.env.local` обновлен. |
