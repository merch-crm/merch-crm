# 📖 Глоссарий Предметной Области MerchCRM

Этот словарь определяет каноническое соответствие между бизнес-терминами (RU) и их представлением в коде (EN). 

> [!IMPORTANT]
> **Правило ИИ:** При написании кода, комментариев или документации используй только термины из колонки **Code (EN)**.

## 📦 Модуль: Заказы и Продажи

| Термин (RU) | Code (EN) | Описание |
| :--- | :--- | :--- |
| **Заказ** | `Order` | Основная сущность сделки с клиентом. |
| **Позиция заказа** | `OrderItem` | Конкретный товар или услуга внутри заказа. |
| **Статус заказа** | `orderStatus` | Состояние жизненного цикла: `new`, `design`, `production`, `done` и т.д. |
| **Категория заказа** | `orderCategory` | Тип работ: `print` (печать), `embroidery` (вышивка), `merch`. |
| **Приложение** | `Attachments` | Файлы, прикрепленные к заказу (ТЗ, макеты). |
| **Срочный** | `isUrgent` | Флаг приоритетного выполнения. |

## 🏭 Модуль: Производство

| Термин (RU) | Code (EN) | Описание |
| :--- | :--- | :--- |
| **Статус печати** | `stagePrintStatus` | Текущее состояние этапа печати для позиции. |
| **Макет / Дизайн** | `Design` | Визуальное решение, требующее утверждения (`approved`, `revision`). |
| **Тип нанесения** | `applicationType` | Технология: DTF, Шелкография, Вышивка и т.д. |
| **Брак** | `defectReason` | Причина списания: `equipment_failure`, `human_error`. |

## 🏦 Модуль: Склад и Финансы

| Термин (RU) | Code (EN) | Описание |
| :--- | :--- | :--- |
| **Товар склада** | `inventoryItem` | Физическая единица на складе (футболка, упаковка). |
| **Локация** | `storageLocation` | Место хранения: `warehouse`, `production`, `office`. |
| **Транзакция** | `transaction` | Любое движение товара (`stock_in`, `stock_out`, `adjustment`). |
| **Метод оплаты** | `paymentMethod` | Способ: `cash`, `bank`, `online`, `account`. |

## 👥 Общие Сущности

| Термин (RU) | Code (EN) | Описание |
| :--- | :--- | :--- |
| **Клиент** | `Client` | Контрагент (B2B или B2C). |
| **Менеджер** | `Manager` | Ответственный сотрудник за заказ. |
| **Промокод** | `Promocode` | Скидочный инструмент. |
