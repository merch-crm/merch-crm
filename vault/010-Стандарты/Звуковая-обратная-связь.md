# Звуковая и вибро-обратная связь

## Библиотека
```typescript
import { playSound } from '@/lib/sounds';
```

## Маппинг действий → звуков

| Действие          | SoundType               |
|-------------------|-------------------------|
| Успех             | `notification_success`  |
| Ошибка            | `notification_error`    |
| Создание          | `notification_success`  |
| Удаление          | `client_deleted`        |
| Успешный заказ    | `order_created`         |
| Финансы           | `expense_added`         |
| Задача выполнена  | `task_completed`        |
| Склад / приход    | `stock_replenished`     |

## Правила
- Массовые операции — один звук после завершения всех.
- Не вызывать `navigator.vibrate()` напрямую.
- `playSound()` управляет и звуком, и вибрацией.
