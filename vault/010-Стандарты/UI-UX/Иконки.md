# 🎨 Стандарты Иконок

В проекте MerchCRM установлен единый стандарт использования иконок для обеспечения консистентности интерфейса и минимизации размера бандла.

## 🏆 Единый Стандарт: Lucide React

Все иконки в приложении **должны** импортироваться только из библиотеки `lucide-react`. 

### Почему Lucide?
1.  **Tree-shaking**: Поддерживает эффективное удаление неиспользуемого кода.
2.  **Стиль**: Чистый, современный stroke-дизайн, идеально сочетающийся с `shadcn/ui`.
3.  **Производительность**: Легковесная и стабильная библиотека.

## 🚫 Запрещенные библиотеки
Использование следующих библиотек **строго запрещено** и будет блокироваться на этапе сборки:
- `@phosphor-icons/react`
- `@tabler/icons-react`
- `react-icons` (весь бандл)

## 📏 Правила использования

### Импорт
Рекомендуется использовать деструктуризацию для удобства tree-shaking (Next.js оптимизирует это автоматически):
```typescript
import { User, Settings, Check } from "lucide-react";
```

### Размеры и стилизация
- **Стандартный размер**: 16px или 20px (в зависимости от контекста).
- **Stroke Width**: Рекомендуется `2px` (по умолчанию в Lucide).
- **Цвет**: Используйте Tailwind классы для управления цветом (`text-foreground`, `text-primary` и т.д.).

### Пример
```tsx
<Button>
  <User className="mr-2 h-4 w-4" />
  Профиль
</Button>
```
