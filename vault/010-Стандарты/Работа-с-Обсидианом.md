# Регламент работы в Obsidian Vault

Этот регламент устанавливает стандарты структуры, именования и взаимодействия с базой знаний MerchCRM.

## 1. Структура папок (Luhmann Style)

Хранилище разделено на тематические блоки с фиксированной нумерацией:

- **000-Навигация**: Главные входы, оглавления, дорожные карты, чек-листы.
- **010-Стандарты**: Технические регламенты, правила дизайна и кодинга.
- **020-Архитектура**: ADR (Architectural Decision Records), инфраструктурные схемы.
- **030-Модули**: Спецификации функциональных модулей (разбиты по отделам).
- **040-Решения**: Описания реализованных алгоритмов, интеграций и сложных функций.
- **050-Сессии**: Журналы рабочих сессий с ИИ (ежедневные отчеты).
- **060-Roadmap**: Планы развития, вехи проекта.
- **070-Шаблоны**: Заготовки для новых файлов (Модули, Баг-репорты, ADR).

## 2. Стандарты именования

Для обеспечения чистоты и автоматической сортировки файлов применяются следующие правила:

### 2.1. Файлы
- **Стиль**: `Название-Файла.md` (Title-Case с дефисами).
- **Запрещено**: Использовать `ALL-CAPS`, `нижние_подчеркивания`, `пробелы`.
- **Язык**: Русский (для смысловых названий), Английский (для терминов или названий инструментов).

### 2.2. Папки
- **Стиль**: `###-Название-Папки`.
- **Подпапки (в Модулях)**: `##-Название-Отдела` (например, `01-Продажи-и-Клиенты`).

## 3. Организация Модулей (030-Модули)

Все модули функционала должны находиться в одной из папок отделов:
1. **01-Продажи-и-Клиенты**: CRM, Заказы, Калькуляторы, Промокоды.
2. **02-Производство-и-Склад**: Производство, Печать, Склад, Остатки.
3. **03-Персонал-и-HR**: Сотрудники, Присутствие, Мотивация.
4. **04-Система-и-Финансы**: Ядро, Финансы, Логи, Настройки.

## 4. Жизненный цикл документа

1. Новый документ создается **обязательно** из шаблона (`070-Шаблоны`).
2. При создании фичи файл переводится из статуса `draft` в `plan`.
3. После реализации функционала и тестов файл переходит в статус `active`.
4. Любые серьезные изменения в коде должны быть сначала отражены в соответствующей документации Obsidian.
