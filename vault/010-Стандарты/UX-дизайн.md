# UX и Дизайн (Modern Industrial Craft)

## 1. Основы
- **Сетка**: Шаг 4px.
- **Отступы**: Стандарт `gap-3` (12px).
- **Радиусы**: 16px для карточек (XL), 12px для диалогов (L), 6px для инпутов (S).

## 2. Эстетика
Использовать «стеклянный» эффект (glassmorphism) для фоновых панелей.
