# UX/UI Дизайн (Lumin-Apple v3.0)

MerchCRM Unified v3.0 базируется на эстетике **Lumin-Apple**: сочетании премиальной лаконичности интерфейсов Apple с динамичной «светящейся» акцентной палитрой.

## 1. Основы (Система токенов)
- **Сетка**: 8px (базовый шаг: 4px для микро-отступов).
- **Скругления (Border Radius)**:
	- **Outer (Карточки/Контейнеры)**: 24px (`rounded-3xl`). Это создает «мягкий» и современный вид.
	- **Inner (Кнопки/Инпуты)**: 14px (`rounded-xl`). Гармонично вписываются во внешние блоки.
- **Цвета**:
	- **Primary**: `#5d00ff` — насыщенный фиолетовый. Энергия и фокус.
	- **Background**: `#f8f9fa` (Light) / `#0a0a0a` (Dark).
	- **Surface**: Белый с легкой прозрачностью (85%+) и размытием.

## 2. Визуальные эффекты
- **Glassmorphism**: Панели должны иметь `backdrop-blur-xl` и тонкую границу `border-white/10`.
- **Градиенты**: Использовать мягкие градиенты для кнопок и акцентов (например, с переходом в более светлый фиолетовый).
- **Тени (Elevations)**: Мягкие, многослойные тени (Ambient Shadows), избегать резких краев.
- **Типографика**: Приоритет — **Geist** или **Inter**. Четкий фокус на иерархии через веса (Medium для заголовков).

## 3. Интерактивность
- **Spring-анимации**: Все модальные окна и переходы должны использовать физические пружинные анимации (без резких рывков).
- **Микро-взаимодействия**: Ховеры (Hover states) всегда должны менять масштаб (`scale-105`) или прозрачность, давая мгновенный отклик.
- **Индикация состояний**: 
	- Успех: `#10b981` (Emerald).
	- Ошибка: `#ef4444` (Rose).
	- Предупреждение: `#f59e0b` (Amber).

## 4. Конструкция компонентов
- **Header**: Всегда плавающий (sticky) с эффектом размытия заднего плана.
- **Sidebar**: Минималистичный, без жестких границ, с акцентным фоном активного пункта.
- **Input**: Высота 44px-48px для удобства клика (Touch-friendly).
