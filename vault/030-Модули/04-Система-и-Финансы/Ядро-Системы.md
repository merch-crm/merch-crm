# Ядро Системы (System Core)

Инфраструктурный слой: аудит-логи, безопасность, системные настройки, уведомления, API-ключи, редактор макетов, брендинг документов и шрифты.

---

## 1. Таблица `audit_logs`

> **Файл:** `lib/schema/system.ts`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid NOT NULL` | ID (часть composite PK) |
| `userId` | `uuid FK → users` | Кто выполнил действие |
| `action` | `text NOT NULL` | Тип действия |
| `actionCategory` | `text` | Категория действия |
| `entityType` | `text NOT NULL` | Тип сущности |
| `entityId` | `uuid NOT NULL` | ID сущности |
| `details` | `jsonb` | Подробности |
| `createdAt` | `timestamp NOT NULL` | Дата |

> ⚠️ **PK:** Composite — `(id, createdAt)` (для партиционирования по времени).

### Индексы
- `audit_logs_user_idx` → `userId`
- `audit_logs_entity_idx` → `(entityType, entityId)`
- `audit_logs_action_idx` → `action`
- `audit_logs_category_idx` → `actionCategory`
- `audit_logs_created_idx` → `createdAt`
