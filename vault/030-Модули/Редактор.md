# Редактор макетов

Модуль визуального конструктора макетов на базе Fabric.js. Позволяет создавать, хранить и экспортировать дизайны для продукции (мерча).

## Схема базы данных

### editor_projects
Основная таблица проектов (сохраненных состояний холста).

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | `uuid` | Первичный ключ |
| `order_id` | `uuid` | Привязка к заказу (FK) |
| `order_item_id` | `uuid` | Конкретная позиция в заказе |
| `design_id` | `uuid` | Привязка к готовому дизайну (FK) |
| `name` | `text` | Название проекта |
| `description` | `text` | Описание |
| `width` | `integer` | Ширина холста (px) |
| `height` | `integer` | Высота холста (px) |
| `canvas_data` | `jsonb` | Состояние холста в формате Fabric.js JSON |
| `thumbnail_path` | `text` | Путь к миниатюре превью |
| `is_template` | `boolean` | Флаг шаблона для повторного использования |
| `is_finalized` | `boolean` | Флаг утвержденной версии |
| `created_by` | `uuid` | Автор проекта (FK) |
| `created_at` | `timestamp` | Дата создания |
| `updated_at` | `timestamp` | Дата обновления |

### editor_exports
История экспорта файлов (рендеров) из редактора.

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | `uuid` | Первичный ключ |
| `project_id` | `uuid` | Ссылка на проект (FK) |
| `filename` | `text` | Имя файла |
| `format` | `text` | Формат (`png`, `jpeg`, `svg`, `pdf`) |
| `width` | `integer` | Ширина рендера |
| `height` | `integer` | Высота рендера |
| `size` | `integer` | Размер файла в байтах |
| `path` | `text` | Путь к файлу в хранилище |
| `has_watermark` | `boolean` | Был ли наложен водяной знак |
| `quality` | `integer` | Качество сжатия (для JPEG) |
| `created_by` | `uuid` | Кто выполнил экспорт (FK) |
| `created_at` | `timestamp` | Дата экспорта |

## Логика и связи

- **Связь с Заказами**: Проект может создаваться «под заказ», тогда он получает `order_id`.
- **Шаблоны**: Если `is_template = true`, проект доступен для выбора при создании новых макетов.
- **Хранение**: Сами файлы превью и экспорта хранятся в S3-совместимом хранилище, в БД записываются только пути.
- **Интеграция с Дизайнами**: Утвержденные макеты могут конвертироваться в записи модуля [[Дизайн]].
