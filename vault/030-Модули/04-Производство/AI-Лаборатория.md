# AI-Лаборатория

## 1. Описание (Goal)
Генерация дизайнов с помощью искусственного интеллекта. Позволяет дизайнерам создавать макеты и принты на основе текстовых промтов или загруженных референсов, ускоряя процесс разработки дизайнов для мерча.

## 2. Функциональность
- [x] Генерация дизайнов по текстовому описанию (OpenAI DALL-E)
- [x] Удаление фона и апскейл изображений (Replicate)
- [x] Интеграция с модулем «Дизайн» для быстрого применения

## 🔗 Связи и Интеграции
- **Дизайн**: [[030-Модули/05-Дизайн/Дизайн|Мастерская]] — передача сгенерированных принтов в макеты.
- **Производство**: [[030-Модули/04-Производство/Производство|Цех]] — предпросмотр ИИ-результатов на готовых изделиях.
- **Хранилище**: [[030-Модули/03-Склад/Хранилище|Склад]] — сохранение сгенерированных ассетов.
