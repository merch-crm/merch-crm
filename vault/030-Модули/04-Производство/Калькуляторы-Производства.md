# Калькуляторы Производства

## 1. Описание (Goal)
Система калькуляторов себестоимости для 7 технологий нанесения. Каждый калькулятор учитывает расходные материалы, амортизацию оборудования, трудозатраты и наценку для расчёта финальной стоимости изделия.

## 2. Поддерживаемые технологии

| Технология | Директория | Ключевые параметры |
|-----------|-----------|-------------------|
| **DTF** (прямая печать на плёнку) | `dtf/` | размер принта, расход чернил, плёнки, порошка |
| **DTG** (прямая печать на ткань) | `dtg/` | площадь печати, тип ткани, белая подложка |
| **Вышивка** | `embroidery/` | количество стежков, плотность, тип ниток |
| **Шёлкография** | `silkscreen/` | количество цветов, тираж, тип краски |
| **Сублимация** | `sublimation/` | площадь покрытия, тип носителя |
| **Термотрансфер** | `thermotransfer/` | размер, тип трансфера |
| **UV-DTF** | `uv-dtf/` | площадь, слои лака |

## 3. Общая архитектура
```
calculators/
├── page.tsx              — навигация между калькуляторами
├── layout.tsx            — общий лэйаут
├── types.ts              — общие типы (12KB)
├── print-application-types.ts — типы нанесения (12KB)
├── placements/           — зоны размещения
├── history/              — история расчётов
├── components/           — общие компоненты
├── hooks/                — общие хуки
├── lib/                  — утилиты
├── actions/              — серверные действия
└── [dtf|dtg|...]/        — калькуляторы по технологиям
```

## 4. Функциональность
- [x] Расчёт себестоимости с учётом всех расходников
- [x] Визуализация размещения принта (SchematicRollVisualizer для DTF)
- [x] История расчётов с сохранением параметров
- [x] Зоны размещения (placements) для разных типов изделий
- [x] Общие компоненты: виджеты, формы, превью
