# 🏭 Производство
Сердце MerchCRM, отвечающее за преобразование цифровых макетов в физические изделия. Включает управление печатными линиями, контроль качества и расчет технологических процессов.

## 📁 Состав модуля
```dataview
LIST
FROM "030-Модули/04-Производство"
WHERE file.name != this.file.name
SORT file.name ASC
```

## 🔗 Связи и Интеграции
- **Заказы**: [[030-Модули/01-Заказы/Заказы|Очередь Производства]] — получение ТЗ и сроков.
- **Дизайн**: [[030-Модули/05-Дизайн/Дизайн|Макеты]] — загрузка файлов для печати.
- **Склад**: [[030-Модули/03-Склад/Склад|Расходные материалы]] — автоматическое списание краски/пленки/нитей.

## 🛠 Техническая реализация
- **БД**: `lib/schema/production.ts` — таблицы `production_runs`, `print_jobs`.
- **Логика**: `lib/services/production-engine.ts`.
- **UI**: `app/(main)/production/` — дашборд мастера цеха.
