# Отчёты Персонала

## 1. Описание (Goal)
Формирование отчётов по сотрудникам: выработка, посещаемость, эффективность. Используется руководством для принятия кадровых решений и контроля KPI.

## 2. Функциональность
- [x] Генерация отчётов по периодам.
- [x] Фильтрация по отделам и сотрудникам.
- [x] Сравнение показателей производительности.
- [x] Экспорт данных (запланировано).
