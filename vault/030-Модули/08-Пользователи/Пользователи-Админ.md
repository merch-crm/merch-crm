# Пользователи (Админ)

## 1. Описание (Goal)
Управление учётными записями пользователей из админ-панели. CRUD пользователей, назначение ролей и отделов, блокировка/разблокировка, сброс паролей.

## 2. Функциональность
- [x] Список пользователей с поиском и фильтрацией
- [x] Создание и редактирование учётных записей
- [x] Назначение ролей и отделов
- [x] Блокировка / сброс 2FA / сброс пароля

## 3. Техническая реализация
**Файлы:**
- `app/(main)/admin-panel/users/` — модуль пользователей
- `app/(main)/admin-panel/actions/` — серверные действия
