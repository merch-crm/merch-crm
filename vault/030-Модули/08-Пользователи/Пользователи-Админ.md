# 👤 Управление Пользователями (Админ)

## 1. Описание (Goal)
Интерфейс администратора для управления учетными записями всех сотрудников системы. Позволяет создавать новых пользователей, редактировать существующие профили, назначать роли и отделы.

## 2. Функциональность
- [x] Таблица пользователей с пагинацией и поиском.
- [x] Создание нового пользователя через диалоговое окно.
- [x] Редактирование прав, ролей и персональных данных.
- [x] Статистика активности и типов аккаунтов.
- [x] Блокировка и удаление пользователей.

## 🛠 Техническая реализация
- **UI**: 
  - `app/(main)/admin-panel/users/page.tsx` — Основной реестр.
  - `app/(main)/admin-panel/users/users-table.tsx` — Таблица пользователей.
- **БД**: `lib/schema/users.ts` — Таблицы `users`, `roles`, `departments`.
- **Логика**: `app/(main)/admin-panel/users/hooks/` — CRUD операции.
