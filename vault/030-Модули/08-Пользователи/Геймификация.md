# Геймификация (Achievements)

Система ачивок (бейджей) сотрудников с баллами и скрытыми достижениями.

---

## 1. Таблица `achievements`

> **Файл:** `lib/schema/achievements.ts`

Справочник достижений.

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `name` | `varchar(255) NOT NULL` | Название |
| `description` | `text NOT NULL` | Описание |
| `icon` | `varchar(100)` | Lucide icon name |
| `points` | `integer DEFAULT 0 NOT NULL` | Баллы |
| `code` | `varchar(100) NOT NULL UNIQUE` | Код (`top_packer_weekly`) |
| `badgeUrl` | `text` | Картинка/бейдж |
| `isHidden` | `boolean DEFAULT false NOT NULL` | Скрытая до получения |
| `isActive` | `boolean DEFAULT true NOT NULL` | Активна |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

---

## 2. Таблица `user_achievements`

M:N связь пользователь ↔ ачивка.

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `userId` | `uuid FK → users (CASCADE) NOT NULL` | Сотрудник |
| `achievementId` | `uuid FK → achievements (CASCADE) NOT NULL` | Достижение |
| `achievedAt` | `timestamp NOT NULL` | Когда получено |
| `details` | `jsonb` | Доп. данные ("Упаковано 500 шт.") |
| `createdAt` | `timestamp NOT NULL` | Создано |

### Индексы
- Composite: `(userId, achievementId)` → дедупликация
