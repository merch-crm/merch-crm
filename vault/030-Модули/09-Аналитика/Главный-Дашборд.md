# Главный Дашборд

## 1. Описание (Goal)
Стартовая страница CRM. Агрегирует ключевые метрики: активные заказы, загрузка производства, финансовые показатели, последние действия. Персонализируется в зависимости от роли пользователя.

## 2. Функциональность
- [x] Клиентский дашборд (`dashboard-client.tsx`, 10KB)
- [x] Глобальный поиск (`search-actions.ts`, 12KB)
- [x] Уведомления (`notifications-actions.ts`, 5KB)
- [x] Отмена действий (`undo-actions.ts`, 3KB)
- [x] Общие действия (`actions.ts`, 8KB)
- [x] Общие хуки (`hooks/`)
- [x] Общие компоненты (`components/`)

## 3. Техническая реализация
**Файлы:**
- `app/(main)/dashboard/page.tsx` — серверная страница
- `app/(main)/dashboard/dashboard-client.tsx` — клиентский дашборд
- `app/(main)/dashboard/search-actions.ts` — глобальный поиск
- `app/(main)/dashboard/actions.ts` — общие действия
