# Коммуникации (Communications)

Омниканальный мессенджер: каналы → диалоги → сообщения + шаблоны.

---

## 1. TypeScript-константы (не pgEnum!)

> **Файл:** `lib/schema/communications.ts`

Определены как `as const` массивы, **не pgEnum**:

| Константа | Значения |
|---|---|
| `channelTypes` | `telegram`, `instagram`, `vk`, `whatsapp`, `email`, `sms` |
| `conversationStatuses` | `active`, `archived`, `blocked` |
| `messageDirections` | `inbound`, `outbound` |
| `messageTypes` | `text`, `image`, `file`, `voice`, `sticker` |
| `messageStatuses` | `pending`, `sent`, `delivered`, `read`, `failed` |
| `templateCategories` | `greeting`, `info`, `closing`, `promo` |

Также экспортируются `channelTypeLabels`, `channelTypeColors`, `channelTypeIcons` и `templateCategoryLabels`.

---

## 2. Таблица `communication_channels`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `name` | `text NOT NULL` | Название |
| `channelType` | `text NOT NULL` | Тип канала ($type\<ChannelType\>) |
| `icon` | `text` | Иконка |
| `color` | `text` | Цвет |
| `isActive` | `boolean DEFAULT true` | Активен |
| `config` | `jsonb DEFAULT {}` | Конфигурация (API keys, webhooks) |
| `createdAt` | `timestamp(tz)` | Создано |

> ⚠️ Timestamps с `withTimezone: true`. Нет `updatedAt`.

---

## 3. Таблица `client_conversations`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `clientId` | `uuid FK → clients (CASCADE) NOT NULL` | Клиент |
| `channelId` | `uuid FK → communicationChannels` | Канал |
| `channelType` | `text NOT NULL` | Тип канала |
| `externalChatId` | `text` | ID в Telegram/IG/VK |
| `status` | `text DEFAULT 'active'` | Статус |
| `unreadCount` | `integer DEFAULT 0` | Непрочитанные |
| `lastMessageAt` | `timestamp(tz)` | Последнее сообщение |
| `lastMessagePreview` | `text` | Превью |
| `assignedManagerId` | `uuid FK → users` | Менеджер |
| `metadata` | `jsonb DEFAULT {}` | Метаданные |
| `createdAt` / `updatedAt` | `timestamp(tz)` | Даты |

## 4. Таблица `conversation_messages`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `conversationId` | `uuid FK → clientConversations (CASCADE) NOT NULL` | Диалог |
| `direction` | `text NOT NULL` | inbound / outbound |
| `messageType` | `text DEFAULT 'text'` | Тип сообщения |
| `content` | `text` | Текст |
| `mediaUrl` | `text` | URL медиа |
| `mediaType` | `text` | Тип медиа |
| `externalMessageId` | `text` | ID в мессенджере |
| `status` | `text DEFAULT 'sent'` | Статус доставки |
| `sentById` | `uuid FK → users` | Отправитель (менеджер) |
| `sentAt` | `timestamp(tz)` | Отправлено |
| `deliveredAt` | `timestamp(tz)` | Доставлено |
| `readAt` | `timestamp(tz)` | Прочитано |
| `createdAt` | `timestamp(tz)` | Создано |
| `metadata` | `jsonb DEFAULT {}` | Метаданные |

## 5. Таблица `message_templates`

|---|---|---|

## 6. Relations

```
communicationChannels → conversations (many)
clientConversations → client (one), channel (one), assignedManager (one), messages (many)
conversationMessages → conversation (one), sentBy (one)
messageTemplates → createdBy (one)
```
