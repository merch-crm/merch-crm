# API-Ключи
Управление ключами доступа к API MerchCRM. Позволяет создавать, отзывать и ротировать ключи для внешних интеграций.

## 🔗 Связи и Интеграции
- **Безопасность**: [[030-Модули/07-Админ-Панель/Роли-и-Права|Роли]] — управление правами доступа.
- **Логи**: [[030-Модули/07-Админ-Панель/Аудит-Действий|Аудит]] — фиксация запросов к API.

## 🛠 Техническая реализация
- **Статус**: В разработке (Planned).
- **Интерфейс**: Будет расположен в `app/(main)/dashboard/settings/api/`.
- **Логика**: Ожидается в `lib/actions/integrations/`.
