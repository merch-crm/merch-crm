# Роли и Права (RBAC)

## 1. Описание (Goal)
Управление ролями и правами доступа (Role-Based Access Control). Каждому пользователю назначается роль, определяющая доступ к модулям, действиям и данным.

## 2. Функциональность
- [x] CRUD ролей с назначением разрешений
- [x] Привязка ролей к пользователям
- [x] Проверка прав на уровне Server Actions и middleware

## 3. Техническая реализация (Implementation)
**Файлы:**
- `app/(main)/admin-panel/roles/` — управление ролями
- `app/(main)/admin-panel/roles-departments/` — связка ролей и отделов
