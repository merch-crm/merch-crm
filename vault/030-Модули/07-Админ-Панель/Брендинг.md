# Брендинг

## 1. Описание (Goal)
Настройка корпоративного стиля CRM: логотип, фавикон, цвета, название компании. Изменения применяются ко всему интерфейсу.

## 2. Функциональность
- [x] Загрузка логотипа и фавикона (`icon-manager.tsx`, 21KB)
- [x] Форма настроек бренда (`branding-form.tsx`, 6KB)
- [x] CRUD-операции брендинга (`actions.ts`, 9KB)
- [x] Валидация (`validation.ts`)
- [x] Кеширование настроек в Redis (TTL 900сек)

## 3. Техническая реализация
**Файлы:**
- `app/(main)/admin-panel/branding/page.tsx` — страница
- `app/(main)/admin-panel/branding/branding-form.tsx` — форма
- `app/(main)/admin-panel/branding/icon-manager.tsx` — менеджер иконок
- `app/(main)/admin-panel/branding/actions.ts` — серверные действия
- `app/(main)/dashboard/settings/branding/` — настройки из дашборда
