# API-Ключи

## 1. Описание (Goal)
Управление ключами доступа к API MerchCRM. Позволяет создавать, отзывать и ротировать ключи для внешних интеграций.

## 2. Функциональность
- [x] Создание и удаление API-ключей
- [x] Ротация ключей
- [x] Привязка к пользователям/сервисам

## 3. Техническая реализация (Implementation)
**Файлы:**
- `app/(main)/admin-panel/api-keys/` — модуль API-ключей
