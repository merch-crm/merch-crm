# Промокоды (Promocodes)

Система скидочных промокодов с гибкими ограничениями.

---

## 1. Таблица `promocodes`

> **Файл:** `lib/schema/promocodes.ts`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `name` | `text` | Название |
| `code` | `text NOT NULL UNIQUE` | Код промокода |
| `discountType` | `text NOT NULL` | Тип: percentage, fixed, free_shipping, gift |
| `value` | `decimal(10,2) NOT NULL` | Значение скидки |
| `minOrderAmount` | `decimal(10,2) DEFAULT '0'` | Мин. сумма заказа |
| `maxDiscountAmount` | `decimal(10,2) DEFAULT '0'` | Макс. скидка |
| `startDate` | `timestamp` | Начало действия |
| `expiresAt` | `timestamp` | Истечение |
| `usageLimit` | `integer` | Лимит использований (null = безлимит) |
| `usageCount` | `integer DEFAULT 0 NOT NULL` | Использовано |
| `isActive` | `boolean DEFAULT true NOT NULL` | Активен |
| `adminComment` | `text` | Комментарий админа |
| `constraints` | `jsonb DEFAULT '{}'` | Ограничения: `{ includedProducts: [], excludedCategories: [] }` |
| `createdAt` | `timestamp NOT NULL` | Создано |

> ⚠️ **Нет `updatedAt`**, `createdById`, `maxUsesPerClient`. `discountType` — обычный text, **не enum**.

### Relations
```
promocodes → orders (many)
```
