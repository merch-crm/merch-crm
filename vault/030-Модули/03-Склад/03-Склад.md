# 📦 Складской Учет
Модуль управления товарными остатками, материалами и оборудованием. Обеспечивает прозрачность запасов и готовность производства.

## 📁 Состав модуля
```dataview
LIST
FROM "030-Модули/03-Склад"
WHERE file.name != this.file.name
SORT file.name ASC
```

## 🔗 Связи и Интеграции
- **Заказы**: [[030-Модули/01-Заказы/Заказы|Бронирование]] — резервирование товара под заказ.
- **Производство**: [[030-Модули/02-Производство/Производство|Списание]] — автоматическое списание материалов при печати.
- **Финансы**: [[030-Модули/06-Финансы/Транзакции|Закупки]] — пополнение склада через финансовые операции.

## 🛠 Техническая реализация
- **БД**: `lib/schema/inventory.ts` — таблицы `inventory`, `stock_moves`.
- **Логика**: `lib/services/inventory-manager.ts`.
- **UI**: `app/(main)/inventory/` — панели управления складом.
