# Присутствие (Presence & Biometrics)

Аппаратная интеграция с камерами Xiaomi, распознавание лиц, рабочие сессии, рабочие станции.

---

## 1. Таблица `xiaomi_accounts`

> **Файл:** `lib/schema/presence/hardware.ts`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `xiaomiUserId` | `varchar(50) NOT NULL UNIQUE` | Xiaomi User ID |
| `email` | `varchar(255)` | Email |
| `nickname` | `varchar(255)` | Никнейм |
| `encryptedToken` | `text NOT NULL` | Зашифрованный токен API |
| `region` | `varchar(10) DEFAULT 'cn' NOT NULL` | Регион |
| `isActive` | `boolean DEFAULT true NOT NULL` | Активен |
| `lastSyncAt` | `timestamp` | Последняя синхронизация |
| `createdById` | `uuid FK → users (SET NULL)` | Кто добавил |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

## 7. Таблица `workstations`

> **Файл:** `lib/schema/presence/workstations.ts`

Рабочие станции с геозонами на кадре камеры.

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `name` | `varchar(255) NOT NULL` | Название |
| `description` | `text` | Описание |
| `cameraId` | `uuid FK → cameras (SET NULL)` | Камера |
| `assignedUserId` | `uuid FK → users (SET NULL)` | Закреплённый сотрудник |
| `zone` | `jsonb (DetectionZone)` | Зона обнаружения |
| `color` | `varchar(7) DEFAULT '#3B82F6'` | Цвет |
| `sortOrder` | `integer DEFAULT 0` | Порядок |
| `isActive` | `boolean DEFAULT true` | Активна |
| `requiresAssignedUser` | `boolean DEFAULT false` | Только закреплённый |
| `lastSeenUserId` | `uuid` | Последний обнаруженный (без FK!) |
| `lastSeenAt` | `timestamp` | Последнее обнаружение |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

### Типы зон `DetectionZone`
```typescript
type DetectionZone =
  | { type: 'rect'; x, y, width, height: number }
  | { type: 'polygon'; points: Array<{x, y}> }
  | { type: 'circle'; cx, cy, radius: number }
```
