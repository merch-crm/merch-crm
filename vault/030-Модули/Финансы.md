# Финансы (Finance)

Управление платежами и расходами.

---

## 1. Таблица `payments`

> **Файл:** `lib/schema/finance.ts`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `orderId` | `uuid FK → orders (CASCADE) NOT NULL` | Заказ |
| `amount` | `decimal(10,2) NOT NULL` | Сумма |
| `method` | `paymentMethodEnum NOT NULL` | Способ |
| `isAdvance` | `boolean DEFAULT false NOT NULL` | Аванс |
| `comment` | `text` | Комментарий |
| `status` | `text DEFAULT 'completed' NOT NULL` | Статус: completed, pending, cancelled |
| `referenceNumber` | `text` | Номер платёжного документа |
| `notes` | `text` | Примечания |
| `createdBy` | `uuid FK → users` | Кто провёл |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

> ⚠️ `status` — обычный text, **не enum**. Значения: `completed`, `pending`, `cancelled`.

### Индексы
- `payments_order_idx`, `payments_status_idx`, `payments_created_idx`, `payments_created_by_idx`

## 2. Таблица `expenses`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `amount` | `decimal(10,2) NOT NULL` | Сумма |
| `category` | `expenseCategoryEnum NOT NULL` | Категория |
| `description` | `text` | Описание |
| `date` | `date NOT NULL` | Дата расхода |
| `createdBy` | `uuid FK → users` | Кто создал |
| `createdAt` | `timestamp NOT NULL` | Создано |

> ⚠️ **Нет `updatedAt`**. `date` — тип `date` (не timestamp).

### Индексы
- `expenses_created_by_idx`, `expenses_date_idx`, `expenses_category_idx`, `expenses_created_idx`
