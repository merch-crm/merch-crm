# Калькулятор (Print Calculator)

Движок расчёта стоимости печати: прогрессивные тарифы, раскладка на рулоне, расход материалов.

---

## 1. Таблица `meter_price_tiers`

> **Файл:** `lib/schema/calculators.ts`

Прогрессивная шкала цен за погонный метр (чем больше метров — тем дешевле).

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `applicationType` | `varchar(50) NOT NULL` | Тип печати (dtf, sublimation, dtg) |
| `rollWidthMm` | `integer NOT NULL` | Ширина рулона (мм) |
| `fromMeters` | `decimal(10,2) NOT NULL` | От (метров) |
| `toMeters` | `decimal(10,2)` | До (null = бесконечность) |
| `pricePerMeter` | `decimal(10,2) NOT NULL` | Цена за метр |
| `sortOrder` | `integer DEFAULT 0 NOT NULL` | Порядок |
| `isActive` | `boolean DEFAULT true NOT NULL` | Активен |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

### Constraints
- `UNIQUE(applicationType, rollWidthMm, fromMeters)`

## 3. Таблица `calculator_consumables_settings`

Настройки расхода материалов по типу печати.

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `applicationType` | `varchar(50) NOT NULL UNIQUE` | Тип печати |
| `inkWhitePerM2` | `decimal(10,2)` | Расход белых чернил (мл/м²) |
| `inkCmykPerM2` | `decimal(10,2)` | Расход CMYK чернил (мл/м²) |
| `powderPerM2` | `decimal(10,2)` | Расход порошка (г/м², DTF) |
| `paperPerM2` | `decimal(10,2)` | Расход бумаги (м²/м², сублимация) |
| `fillPercent` | `integer DEFAULT 80 NOT NULL` | Заполнение (%) |
| `wastePercent` | `integer DEFAULT 10 NOT NULL` | Брак/отходы (%) |
| `config` | `jsonb` | Доп. настройки |
| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

### Структура `config` (JSON)
```typescript
{
  primerPerItem?: number;      // Праймер для DTG (мл/изделие)
  stabilizatorPerM2?: number;  // Стабилизатор для вышивки
  threadPerKStitch?: number;   // Нить на 1000 стежков
  [key: string]: number | undefined;
}
```

| Поле | Тип | Описание |

| `totalAreaM2` | `decimal(10,4) NOT NULL` | Общая площадь (м²) |
| `printsAreaM2` | `decimal(10,4) NOT NULL` | Площадь принтов (м²) |
| `efficiencyPercent` | `decimal(5,2) NOT NULL` | КПД раскладки (%) |
| `pricePerMeter` | `decimal(10,2) NOT NULL` | Цена за метр |
| `printCost` | `decimal(12,2) NOT NULL` | Стоимость печати |
| `placementCost` | `decimal(12,2) NOT NULL` | Стоимость нанесения |
| `materialsCost` | `decimal(12,2) DEFAULT '0' NOT NULL` | Стоимость материалов |
| `totalCost` | `decimal(12,2) NOT NULL` | ИТОГО |
| `avgCostPerPrint` | `decimal(10,2) NOT NULL` | Средняя стоимость за принт |
| `minCostPerPrint` | `decimal(10,2) NOT NULL` | Мин. за принт |
| `maxCostPerPrint` | `decimal(10,2) NOT NULL` | Макс. за принт |
| `consumptionData` | `jsonb (ConsumptionData)` | Расход материалов |
| `orderId` | `uuid FK → orders (SET NULL)` | Привязка к заказу |

| `createdAt` / `updatedAt` | `timestamp NOT NULL` | Даты |

### Интерфейс `ConsumptionData`
```typescript
interface ConsumptionData {
  items: ConsumptionDataItem[];
  totalMaterialsCost?: number;
  threadConsumption?: { totalThreadMeters: number };
  extraColorsCost?: number;
  discountAmount?: number;
  designs?: Array<{ id, name, stitchCount, threadType, density, hasDigitizing, colorsCount }>;
  prints?: Array<{ id, name, type, sizeId, purchasePrice }>;
  settings?: { includeGarments: boolean; isRush: boolean };
  totalItems?: number;
  totalApplications?: number;
  estimatedTime?: number;
  materials?: Record<string, unknown>[];
  materialsCost?: number;
  breakdown?: Array<Record<string, unknown>>;
}
```
