# 📦 Управление Заказами
Центральный модуль системы для ведения коммерческой деятельности: от приема заявки до отгрузки готовой продукции.

## 📁 Состав модуля
```dataview
LIST
FROM "030-Модули/01-Заказы"
WHERE file.name != this.file.name
SORT file.name ASC
```

## 🔗 Связи и Интеграции
- **Клиенты**: [[030-Модули/02-Клиенты/02-Клиенты|Клиенты]] — история заказов контрагента.
- **Склад**: [[030-Модули/03-Склад/03-Склад|Склад]] — резервирование бланков и материалов.
- **Производство**: [[030-Модули/04-Производство/04-Производство|Производство]] — постановка задач на печать/пошив.
- **Дизайн**: [[030-Модули/05-Дизайн/05-Дизайн|Дизайн]] — подготовка макетов.

## 🛠 Техническая реализация
- **БД**: `lib/schema/orders.ts` (экспортирует `orders`, `orderItems`, `orderPayments`).
- **Логика**: `app/(main)/dashboard/orders/actions/core.actions.ts`.
- **Интерфейс**: 
  - `app/(main)/dashboard/orders/` — Реестр заказов.
  - `app/(main)/dashboard/orders/new` — Визард создания нового заказа.
  - `app/(main)/dashboard/orders/[id]` — Детальная карточка заказа.
