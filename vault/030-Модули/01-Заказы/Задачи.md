# Задачи (Tasks)

Система управления задачами с делегированием, автоматическим созданием, зависимостями.

---

## 1. Таблица `tasks`

> **Файл:** `lib/schema/tasks.ts`

| Поле | Тип | Описание |
|---|---|---|
| `id` | `uuid PK` | Идентификатор |
| `title` | `text NOT NULL` | Заголовок |
| `description` | `text` | Описание |
| `status` | `taskStatusEnum DEFAULT 'new' NOT NULL` | Статус |
| `priority` | `taskPriorityEnum DEFAULT 'normal' NOT NULL` | Приоритет |
| `type` | `taskTypeEnum DEFAULT 'general' NOT NULL` | Тип |
| `deadline` | `timestamp NOT NULL` | Дедлайн (**обязательное поле!**) |
| `createdAt` | `timestamp NOT NULL` | Создано |
| `updatedAt` | `timestamp NOT NULL` | Обновлено |
| `completedAt` | `timestamp` | Завершено |
| `creatorId` | `uuid FK → users (CASCADE) NOT NULL` | Создатель |
| `departmentId` | `uuid FK → departments (SET NULL)` | Отдел |
| `delegatedByUserId` | `uuid FK → users (SET NULL)` | Кто делегировал |
| `delegatedAt` | `timestamp` | Дата делегирования |
| `originalAssigneeId` | `uuid FK → users (SET NULL)` | Первоначальный исполнитель |
| `isAutoCreated` | `boolean DEFAULT false NOT NULL` | Автоматически создана |
| `autoCreatedReason` | `text` | Причина автосоздания |
| `autoCreatedSourceType` | `text` | Тип источника |
| `autoCreatedSourceId` | `uuid` | ID источника |

> ⚠️ **Фактические имена:** `creatorId` (не `createdById`), `delegatedByUserId` (не `delegatedById`). **Нет полей:** `orderId`, `parentTaskId`, `startDate`, `estimatedHours`, `actualHours`, `sortOrder`, `tags`, `isDelegated`.

## 11. Бизнес-логика

### Автоматическое создание задач
- CRON `api/cron/tasks/deadlines/route.ts` проверяет приближающиеся дедлайны
- `isAutoCreated=true` + `autoCreatedReason` указывает источник

### Делегирование
1. Менеджер создаёт → `creatorId`
2. Назначает исполнителя → `taskAssignees`
3. Исполнитель делегирует → `delegatedByUserId`, `delegatedAt`, `originalAssigneeId`
