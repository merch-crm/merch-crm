# 🤖 AI-Лаборатория
Интеграция нейросетевых моделей для автоматической генерации принтов, апскейлинга и удаления фона.

## 👥 Функциональность
- **Генерация**: Stable Diffusion / Midjourney API для создания принтов по промпту.
- **Обработка**: Удаление фона (RemoveBG) и векторизация.
- **Апскейл**: Увеличение разрешения макетов для крупноформатной печати.

## 🔗 Связи и Интеграции
- **Дизайн**: [[030-Модули/05-Дизайн/Дизайн|Принты]] — сохранение результатов генерации.
- **Система**: [[030-Модули/07-Админ-Панель/Интеграции|API Keys]] — настройки ключей доступа.
