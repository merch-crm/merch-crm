# 🤖 AI Лаборатория
Интеграция с нейросетями для создания, модификации и улучшения качества изображений для печати.

## 🔗 Связи и Интеграции
- **Дизайн**: [[030-Модули/05-Дизайн/Дизайн|Генерации]] — сохранение результатов в общую базу.
- **Редактор**: [[030-Модули/05-Дизайн/Редактор|Манипуляции]] — удаление фона и апскейл из интерфейса редактора.

## 🛠 Техническая реализация
- **Провайдеры**: OpenAI (DALL-E 3), Replicate (Stable Diffusion, Upscalers).
- **Логика**: `lib/services/ai-service.ts`.
- **API**: `app/api/design/ai/generate/route.ts`.
