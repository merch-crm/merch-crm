# 🖼 UI Референсы
Коллекция визуальных стилей, паттернов и эталонов интерфейса MerchCRM.

## 📚 Источники Дизайна

### 1. UI Kit — Живая библиотека
- **URL**: `/ui-kit`
- **Код**: `app/ui-kit/`
- **Описание**: Базовые компоненты системы (Buttons, Cards, Tables, и др.).

### 2. Страница «Референсы» — Динамическая галерея
- **URL**: `/dashboard/references`
- **Код**: `app/(main)/dashboard/references/`
- **Описание**: Интерактивные примеры сложных блоков и анимаций.

### 3. Локальный архив — Вдохновение
- **Путь**: `vault/Референсы-Дизайн/`
- **Описание**: Коллекция скриншотов и мудбордов.

## 🛠 Техническая реализация
- **UI Kit**: `@/components/ui` — Radix + Tailwind.
- **Ассеты**: `public/assets/branding/` — логотипы и брендинг.
- **Токены**: `tailwind.config.ts` — основная палитра и сетка.
