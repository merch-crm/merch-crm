# Редактор Дизайнов

## 1. Описание (Goal)
Встроенный графический редактор для создания и редактирования дизайнов. Позволяет работать с макетами принтов прямо в CRM без внешних инструментов.

## 2. Функциональность
- [x] Редактирование дизайнов по ID (`[id]/`)
- [x] Загрузка редактора через динамический импорт (`editor-loader.tsx`)
- [x] Серверные действия (`editor/actions/`)
- [x] Кастомные хуки (`editor/hooks/`)

## 3. Техническая реализация
**Файлы:**
- `app/(main)/dashboard/design/editor/page.tsx` — страница
- `app/(main)/dashboard/design/editor/editor-page-client.tsx` — клиент
- `app/(main)/dashboard/design/editor/editor-loader.tsx` — ленивая загрузка
- `app/(main)/dashboard/design/editor/[id]/` — редактирование по ID
- `app/(main)/dashboard/design/editor/components/` — компоненты
