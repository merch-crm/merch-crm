# 🧊 3D Мокапы
Инструментарий для реалистичного наложения принтов на 3D-модели изделий (футболки, худи, кружки).

## 🔗 Связи и Интеграции
- **Дизайн**: [[030-Модули/05-Дизайн/Дизайн|Текстуры]] — использование дизайнов в качестве карт диффузии.
- **Редактор**: [[030-Модули/05-Дизайн/Редактор|Превью]] — мгновенный просмотр изменений в объеме.
- **Склад**: [[030-Модули/03-Склад/Склад|Основы]] — соответствие 3D-моделей реальным артикулам.

## 🛠 Техническая реализация
- **Библиотека**: `three.js`, `@react-three/fiber`, `@react-three/drei`.
- **Модели**: `public/models/` — форматы `.glb` / `.gltf`.
- **Компонент**: `components/design/MockupViewer.tsx`.
