# 3D-Мокапы

## 1. Описание (Goal)
Визуализация дизайнов на товарах в 3D. Использует WebGL (Three.js) для рендеринга футболок, кружек и других товаров с нанесённым принтом. Позволяет клиенту увидеть конечный результат до производства.

## 2. Функциональность
- [x] 3D-вьювер на базе `@react-three/fiber` и `@react-three/drei`
- [x] Текстовые лейблы через `troika-three-text`
- [x] Динамическая загрузка (SSR отключен) для совместимости
- [x] Интеграция с Turbopack через `transpilePackages`
