# 🎨 Дизайн и Визуализация
Центр творческой подготовки заказов, включающий графический редактор, управление макетами и AI-инструменты для генерации принтов.

## 📁 Состав модуля
```dataview
LIST
FROM "030-Модули/05-Дизайн"
WHERE file.name != this.file.name
SORT file.name ASC
```

## 🔗 Связи и Интеграции
- **Заказы**: [[030-Модули/01-Заказы/Заказы|Файлы заказа]] — привязка утвержденных макетов к позициям заказа.
- **Производство**: [[030-Модули/02-Производство/Производство|Печатные файлы]] — передача подготовленных TIFF/PNG в печать.
- **Склад**: [[030-Модули/03-Склад/Склад|Остатки]] — проверка наличия изделий для мокапов.

## 🛠 Техническая реализация
- **БД**: `lib/schema/designs.ts` — таблицы `designs`, `design_versions`.
- **Логика**: `lib/services/design-service.ts`.
- **UI**: `app/(main)/design/` — управление дизайнами.
