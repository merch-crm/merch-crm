# P&L Отчёт (Прибыли и Убытки)

## 1. Описание (Goal)
Отчёт о прибылях и убытках компании. Показывает выручку, себестоимость (COGS), валовую прибыль, накладные расходы, чистую прибыль и маржу за выбранный период.

## 2. Функциональность
- [x] Показатели: totalRevenue, totalCOGS, grossProfit, totalOverhead, netProfit, margin
- [x] Фильтрация по периодам (сегодня / 7д / 30д / 365д / кастомный)
- [x] Клиентская визуализация (`pl-client.tsx`, 6KB)
