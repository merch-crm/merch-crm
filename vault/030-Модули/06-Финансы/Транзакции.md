# Транзакции

## 1. Описание (Goal)
Журнал всех финансовых операций: оплаты от клиентов и расходы. Двойная загрузка данных с разделением на «Поступления» (payments) и «Расходы» (expenses).

## 2. Функциональность
- [x] Два раздела: входящие платежи и расходы
- [x] Параллельная загрузка данных (`Promise.all`)
- [x] Фильтрация по периодам (сегодня / 7д / 30д / 365д / кастомный)
- [x] Полнофункциональный клиент (`transactions-client.tsx`, 20KB)

## 🔗 Связи и Интеграции
- **Заказы**: [[030-Модули/01-Заказы/Заказы|Заказы]] — платежи по сделкам.
- **P&L**: [[030-Модули/06-Финансы/PnL-Отчёт|P&L Отчет]] — расчет прибыли.
