# 🌐 Клиентский Портал
Интерфейс для самостоятельной работы заказчиков: отслеживание статусов, загрузка макетов и история оплат.

## 🔗 Связи и Интеграции
- **Хаб**: [[030-Модули/02-Клиенты/02-Клиенты|Клиенты]] — родительский узел модуля.
- **Заказы**: [[030-Модули/01-Заказы/01-Заказы|Заказы]] — отображение текущих и прошлых покупок.
- **Дизайн**: [[030-Модули/05-Дизайн/Редактор|Конструктор]] — доступ к редактору для кастомизации.
- **Поддержка**: [[030-Модули/02-Клиенты/Поддержка|Чат]] — связь с менеджером.

## 🛠 Техническая реализация
- **Маршрут**: Запланирован как `app/(portal)/` для внешних пользователей.
- **UI**: Будет интегрирован с CRM для обмена данными в реальном времени.

> [!NOTE]
> Не путать с **Порталом сотрудника** ([[[030-Модули/12-Портал-Сотрудника/12-Портал-Сотрудника|12-Портал-Сотрудника]]]), который уже реализован в `app/(staff)/`.
