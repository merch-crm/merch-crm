# 🌐 Клиентский Портал
Интерфейс для самостоятельной работы заказчиков: отслеживание статусов, загрузка макетов и история оплат.

## 🔗 Связи и Интеграции
- **Заказы**: [[030-Модули/01-Заказы/Заказы|Мои Заказы]] — отображение текущих и прошлых покупок.
- **Дизайн**: [[030-Модули/05-Дизайн/Редактор|Конструктор]] — доступ к редактору для кастомизации.
- **Поддержка**: [[030-Модули/02-Клиенты/Поддержка|Чат]] — связь с менеджером.

## 🛠 Техническая реализация
- **Маршрут**: `app/(portal)/` — изолированный интерфейс для внешних пользователей.
- **Авторизация**: `lib/auth/portal-strategy.ts` — доступ по email/паролю или ссылке.
- **UI**: `app/(portal)/dashboard/`.
