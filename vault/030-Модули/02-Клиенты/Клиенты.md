# Клиенты (Clients)

CRM-модуль: управление клиентской базой, воронка продаж, лояльность, RFM-анализ.

---

## 1. TypeScript-константы (clients/main.ts)

### Этапы воронки (`funnelStages`)
| Значение | Label | Цвет |
|---|---|---|
| `lead` | Лид | `#64748B` |
| `first_contact` | Первый контакт | `#3B82F6` |
| `negotiation` | Переговоры | `#F59E0B` |
| `first_order` | Первый заказ | `#10B981` |
| `regular` | Постоянный клиент | `#6366F1` |

### Причины потери (`lostReasons`)
`price` | `competitor` | `quality` | `timing` | `no_need` | `no_response` | `other`

### Индексы (17 штук!)
- `clients_manager_idx`, `clients_archived_idx`, `clients_search_idx` (lastName, firstName, phone)
- `clients_phone_idx`, `clients_email_idx`, `clients_company_idx`, `clients_created_idx`
- `idx_clients_funnel_stage`, `idx_clients_loyalty_level`
- `idx_clients_total_orders_count`, `idx_clients_total_orders_amount`, `idx_clients_last_order_at`
- `idx_clients_rfm_segment`, `idx_clients_rfm_score`
- `idx_clients_city`, `idx_clients_source`, `idx_clients_client_type`

### Relations
```
clients → manager (one: users)
clients → orders (many)
clients → contacts (many: clientContacts)
clients → loyaltyLevel (one: loyaltyLevels)
```
