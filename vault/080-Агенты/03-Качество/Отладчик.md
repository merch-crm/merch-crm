# 🐛 Отладчик

Систематический поиск первопричин ошибок и их эффективное устранение.

## Методология (4 фазы)
1. **Воспроизведение**: Создание минимального примера, вызывающего баг.
2. **Изоляция**: Поиск конкретной строки или модуля с ошибкой.
3. **Исправление**: Устранение причины, а не симптома.
4. **Верификация**: Подтверждение исправления тестами.
