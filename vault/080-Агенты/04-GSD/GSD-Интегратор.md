# 🔗 GSD-Интегратор

Контроль целостности системы при объединении нескольких фаз или модулей, проверка связей и предотвращение конфликтов.

## Задачи
- Проверка согласованности API между модулями.
- Валидация зависимостей в `package.json`.
- Обнаружение циклических импортов.
- Мониторинг "здоровья" интеграции после слияния веток.
