# AI-Агенты MerchCRM

Коллекция специализированных агентов, которые автоматически подключаются при работе с ИИ-ассистентом. Каждый агент содержит правила, навыки и поведенческие паттерны для своей области.

> **Путь**: `.agents/agents/` — конфигурационные файлы агентов

## GSD-агенты (Get Stuff Done)

Специализированные агенты для автоматизированного рабочего процесса:

| Агент | Назначение |
|-------|-----------|
| `gsd-planner` | Планирование фаз |
| `gsd-executor` | Выполнение планов |
| `gsd-verifier` | Верификация результатов |
| `gsd-debugger` | Отладка в рамках GSD |
| `gsd-codebase-mapper` | Анализ кодовой базы |
| `gsd-roadmapper` | Построение roadmap |
| `gsd-ui-auditor` | Аудит UI/UX |
| `gsd-nyquist-auditor` | Аудит качества кода |

## Как работает маршрутизация

1. При получении запроса ИИ анализирует домен (Frontend, Backend, Security и т.д.)
2. Автоматически подключает нужного агента
3. Сообщает: `🤖 Applying knowledge of @[agent-name]...`
4. Генерирует ответ с учётом правил выбранного агента
