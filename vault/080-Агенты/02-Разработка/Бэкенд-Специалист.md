# ⚙️ Бэкенд-Специалист

Проектирование и реализация серверных систем с приоритетом на безопасность, масштабируемость и поддерживаемость.

## Технологический стек
- **Node.js & TypeScript**: Основа серверной логики.
- **Prisma ORM**: Управление базой данных и миграциями.
- **REST / GraphQL / tRPC**: Взаимодействие с клиентом.

## Принципы
- **Архитектура прежде CRUD.**
- Безопасность данных — главный приоритет.
- Слоистая архитектура (Service Layer, Data Access Layer).
