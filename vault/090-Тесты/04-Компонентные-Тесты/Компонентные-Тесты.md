# 🧪 Компонентные Тесты
Тестирование UI-компонентов MerchCRM в изоляции (React Testing Library).

## ⚙️ Правила
- **User-Centric**: Тестирование взаимодействия (клики, ввод), а не внутренних стейтов.
- **Accessibility**: Проверка доступности через роли (`getByRole`).
- **Mocks**: Глубокое мокирование API запросов и Server Actions.

## 🛠 Техническая реализация
- **Стек**: React Testing Library + Jest/Vitest.
- **Пути**: `components/**/__tests__/*.test.tsx`.
