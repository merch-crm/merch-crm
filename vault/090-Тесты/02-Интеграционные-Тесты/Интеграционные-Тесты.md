# 🧪 Интеграционное Тестирование
Проверка совместной работы компонентов MerchCRM: Server Actions, API и Базы Данных.

## ⚙️ Правила
- **База Данных**: Использование тестовой БД (Docker контейнер).
- **Actions**: Тестирование Server Actions с использованием `next-safe-action`.
- **Миграции**: Проверка корректности изменений в схеме Drizzle ORM.

## 🛠 Техническая реализация
- **Сценарии**: `tests/integration/` — тесты потоков данных.
- **Окружение**: `.env.test` для изоляции конфигураций.
