## 🛠️ Инструментарий Разработчика

Для работы с большим объемом кода внедрены следующие инструменты:

### 1. Умный поиск (Smart Find)
**Команда:** `npm run find -- "запрос"`
**Описание:** Скрипт-обертка (`scripts/find.sh`), который автоматически игнорирует тяжелые директории (`node_modules`, `.next`, `dist`, `vault`) и использует `ripgrep` (rg) для мгновенного поиска. Если `rg` не установлен, переключается на оптимизированный `grep`.

### 2. SSH-Туннель (Production Access)
**Команда:** `npm run dev:ssh`
**Описание:** Автоматизация через `autossh` для проброса портов к БД и Redis в Docker-контейнеры на удаленном сервере.
