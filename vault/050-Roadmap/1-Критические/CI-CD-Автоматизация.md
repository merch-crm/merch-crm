# 🚀 CI/CD Автоматизация и Миграции

## Цель
Создание надежного и предсказуемого процесса деплоя MerchCRM, полностью исключающего риски потери данных из-за ручных миграций.

## Что реализовано ✅
- **CI пайплайн (`ci.yml`)**: Автоматическая проверка каждого Pull Request (Lint, Types, Tests).
- **CD пайплайн (`deploy.yml`)**: Автоматическая сборка Docker-образов и деплой на сервер при мердже в `main`.
- **Контейнер-Мигратор**: Отдельный сервис в Docker, безопасно применяющий SQL-миграции Drizzle перед запуском приложения.
- **Интеграция с GitHub Registry (GHCR)**: Безопасное хранение версионных образов.

## Технические подробности
- **Инструменты**: GitHub Actions, Docker, Drizzle ORM, `tsx`.
