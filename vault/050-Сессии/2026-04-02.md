## 🐛 Найденные и решённые проблемы

| Проблема | Причина | Решение |
| :--- | :--- | :--- |
| OOM (Exit 137) при сборке | Нехватка RAM и отсутствие Swap | Лимиты в Docker + Создание SWAP-файла |
| 404 Favicon | Отсутствие файла и некорректные метаданные | `public/favicon.ico` + правка `layout.tsx` |
| Auth Error (Production) | Динамические импорты в Better Auth | Замена на статические `import` |
| Password Auth Failed | Рассинхрон паролей local/prod | Принудительный `ALTER USER` |
| CSP noise on API | Отсутствие исключений для API-роутов | Middleware: `(?!api|...)` и удаление `x-nonce` |
| Non-paginated RedisMock | `SCAN` возвращал всё за один раз | Полноценный `scan` с курсорами на именах ключей |
