# 3D Profile Badge Lanyard

## Описание
Внедрить интерактивный 3D бейдж (Lanyard) на страницу профиля пользователя. Компонент должен использовать физику (@react-three/rapier) для реалистичного раскачивания при взаимодействии мыши или touch-событиях.

## Задачи
- [x] Реализовать базовый 3D компонент Lanyard (выполнено)
- [x] Настроить физические соединения RopeJoint/SphericalJoint (выполнено)
- [ ] Интегрировать `Lanyard3D` в лейаут профиля `app/(main)/dashboard/profile/page.tsx`
- [ ] Оптимизировать загрузку ассетов (card.glb, lanyard.png) 
- [ ] Настроить кастомную текстуру с именем пользователя для бейджа

## Метрики успеха  
- 3D-движок подгружается лениво (next/dynamic) и не блокирует начальную отрисовку LCP.
- Бейдж реагирует на курсор (drag & drop) плавно без потерь FPS.
- Интерфейс профиля выглядит премиально (без дефолтного синего/фиолетового цвета, используя индустриальный дизайн).
