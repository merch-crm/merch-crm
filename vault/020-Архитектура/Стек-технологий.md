# Стек технологий MerchCRM

## Основные компоненты
- **Frontend**: Next.js 16.1.6 (App Router)
- **Styling**: Tailwind CSS v4 (Oxide)
- **State Management**: React 19 Actions + Hooks
- **Database**: PostgreSQL
- **ORM**: Drizzle ORM
- **Auth**: Better Auth (Redis sessions)
- **Cache**: Redis
- **Storage**: S3 (local/remote)
- **Testing**: Vitest, Playwright, MSW
