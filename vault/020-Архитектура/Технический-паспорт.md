## 📦 3. Инфраструктура (Docker)
Проект развернут в Docker. Контейнеры:
1.  `merch-crm` (Node.js/Next.js) — порт 3000 (внутренний)
2.  `merch-crm-db` (PostgreSQL) — только внутренняя сеть Docker (порт НЕ пробрасывается на хост в production)
3.  `merch-crm-redis` (Redis 7) — только внутренняя сеть Docker, данные персистятся в томе `redis_data`
4.  `nginx-proxy` (Nginx Proxy Manager) — порты 80, 81, 443
5.  `merch-go2rtc` (go2rtc) — видеопоток, порты 1984, 8554, 8555
6.  `merch-face-recognition` — распознавание лиц, порт 8001 (GPU)

**Путь к проекту на сервере:** `/root/merch-crm`

> [!IMPORTANT]
> Дефолтные пароли (`postgres`) **удалены**. Все пароли (`POSTGRES_PASSWORD`, `REDIS_PASSWORD`) обязаны быть явно заданы в `.env`.

## 🤖 7. Автоматизация (GitHub Actions & Deploy)
В репозитории настроены Repository Secrets для GitHub Actions. 

**Правило деплоя:**
*   После того как AI выполнит `git push` и сообщит об этом, он обязан дождаться подтверждения от ПОЛЬЗОВАТЕЛЯ (например, фразы «код успешно загружен»).
*   **Только после сообщения пользователя** AI автоматически запускает воркфлоу `/deploy_production` или команду пересборки.
*   **Команда деплоя:** `ssh -i ~/.ssh/antigravity_key root@89.104.69.25 "cd /root/merch-crm && docker compose up -d --build"`
