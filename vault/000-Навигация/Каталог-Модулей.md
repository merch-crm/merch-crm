# 📦 Каталог Модулей

Этот дашборд автоматически индексирует все спецификации модулей из папки `030-Модули`.

> [!TIP]
> Плагин **DataCards** автоматически должен преобразовать эту таблицу в карточки, если в настройках включен соответствующий режим.

```dataview
TABLE 
    статус AS "Статус", 
    прогресс AS "%", 
    ответственный AS "Отв."
FROM "030-Модули"
WHERE статус != null
SORT статус DESC
```

## 🚀 В процессе (In Progress)
```dataview
LIST
FROM "030-Модули"
WHERE статус = "plan" OR статус = "draft"
```

## ✅ Готовые (Active)
```dataview
LIST
FROM "030-Модули"
WHERE статус = "active"
```
