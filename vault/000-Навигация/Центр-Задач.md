# 🎯 Центр Задач

> [!NOTE]
> Глобальный агрегатор всех активных задач по проекту из всех разрабатываемых модулей.
> Использует плагин **Tasks**. Для вывода задач, пометьте список задач галочками (чекбоксами).

## 🚨 Высокий приоритет и Просроченные
> [!WARNING] Задачи, которые горят. Фокус №1.
```tasks
not done
due before today OR priority is above normal
sort by due
```

## 🟡 Текущий бэклог (В работе)
> [!TIP] Задачи без даты, отсортированные по статусу.
```tasks
not done
no due date
priority is normal
sort by path
```

## 🟢 Выполненные (Недавние)
> [!SUCCESS] Лог последних закрытых задач.
```tasks
done
sort by done reverse
limit 10
```
