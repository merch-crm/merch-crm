# Известные проблемы и Технический долг

Здесь фиксируются все "костыли", баги и проблемы безопасности, которые требуют решения.

## 🔴 Критические (High Priority)
- **Утечка секретов**: В файле `.agents/workflows/start_dev_session.md` хранятся открытые пароли БД и IP. 
    - *Решение*: Перейти на использование `.env.local` или GitHub Secrets. Ротировать текущие пароли.
- **Безопасность Docker**: Образы тегируются только как `:latest`. Нет версионирования и возможности отката.

## 🟡 Средние (Medium Priority)
- **TypeScript `any`**: Обнаружено ~60 использований `any`. 
    - *Файлы*: `permission-shield.tsx`, `audit-client.tsx`, `orders/[id]/page.tsx`, `use-duplicate-check.ts`.
    - *Стандарт*: [[010-Стандарты/TypeScript-строгость]].
- **Отсутствие индексов**: В `lib/schema/orders.ts` не хватает индексов для фильтрации по датам (`createdAt`).

## 🟢 Низкие (Low Priority)
- **Старые README**: В корне проекта много неактуальных README-файлов, которые путают ИИ и пользователя.
- **Звуковые уведомления**: Не на всех страницах реализован `playSound()` при успешных мутациях.
