# 📦 Оптимизация раскладки макетов

Для минимизации отходов пленки (DTF) и другого роллетного материала используется алгоритм упаковки в контейнер.

## Локализация в проекте
* `lib/utils/layout-optimizer.ts`

## Алгоритм
Используется **Greedy Bin Packing** (Жадная упаковка) с предварительной сортировкой по высоте/площади. Это позволяет быстро генерировать эффективные раскладки для производства без значительных вычислительных затрат.

## Перспективы
В будущем возможен переход на более сложные эвристики (например, MaxRects) при увеличении сложности макетов.
