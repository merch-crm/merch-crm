# 🤖 QA-Автоматизатор

Покрытие функциональности автоматическими тестами (Unit, Integration, E2E), создание тестовых сценариев и поддержание тестовой инфраструктуры.

## Задачи
- Разработка E2E тестов с использованием Playwright.
- Написание компонентных и модульных тестов на Vitest + React Testing Library.
- Настройка и поддержка MSW (Mock Service Worker) для имитации API-ответов.
- Интеграция тестов в CI/CD процессы проекта.
- Анализ покрытия кода и поддержка визуальных регрессионных тестов.
