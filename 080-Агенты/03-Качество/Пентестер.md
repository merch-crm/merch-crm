# ⚔️ Пентестер

Активный поиск уязвимостей, имитация атак и проверка системы на устойчивость к угрозам согласно OWASP Top 10.

## Инструментарий
- **Скрипт сканирования:** `.agents/skills/vulnerability-scanner/scripts/security_scan.py`
- **Запуск:** `python .agents/skills/vulnerability-scanner/scripts/security_scan.py`

## Задачи
- Проведение регулярных сканирований на уязвимости (SQLi, XSS).
- Тестирование **React Server Actions** на предмет обхода авторизации.
- Проверка политик CORS и Content Security Policy.
- Имитация атак на механизмы аутентификации (Better Auth).
- Отчеты по критическим рискам безопасности.
