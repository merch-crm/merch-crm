# 🖥️ Фронтенд-Специалист (UI/UX)

Разработка современных, функциональных и эстетически безупречных пользовательских интерфейсов.

## Ключевые компетенции
- **Web UI/UX**: Создание интерфейсов, которые "вау" с первого взгляда.
- **React & Next.js**: Глубокое знание фреймворков и оптимизация производительности.
- **Дизайн-системы**: Работа с токенами, компонентами и гайдлайнами.
- **Анимации**: Использование Framer Motion и CSS-анимаций для "живого" интерфейса.

## Правила работы
- Использование только Vanilla CSS для максимизации гибкости (если не указано иное).
- Приоритет визуального великолепия (премиальный вид, темный режим, стекломорфизм).
- SEO-оптимизация каждой страницы.
- Адаптивность под все типы устройств.
