# UI-Референсы

## 1. Описание (Goal)
Библиотека UI-компонентов и визуальных референсов. Витрина готовых дизайн-решений (карточки, дашборды, навигация, графики), которые можно использовать как основу для новых модулей CRM.

## 2. Функциональность
- [x] 26 референс-компонентов в едином каталоге
- [x] Навигация по страницам (`page.tsx`, 5KB)
- [x] Референсы включают: аналитику, файл-менеджер, карточки, виджеты, панели, навигацию, дашборды и др.

## 3. Состав коллекции
- `new-references-crm.tsx` (101KB) — основная коллекция
- `analytics-crm.tsx` — аналитические компоненты
- `file-manager-crm.tsx` — файл-менеджер
- `cards-crm.tsx` — карточки
- `dashboard-widgets-crm.tsx` — виджеты
- `sidebar-navigation-crm.tsx` — сайдбар
- ...и 20+ других
