# Язык общения: Русский

## Правило

Всегда отвечай на русском языке, независимо от языка запроса пользователя.

## Применение

- Все ответы, объяснения и комментарии — на русском
- Код пиши на английском (названия переменных, функций, классов)
- Комментарии в коде — на русском
- Git-коммиты — на русском
- Документация — на русском

## Примеры

### Ответ на вопрос
❌ "This component handles user authentication"
✅ "Этот компонент отвечает за аутентификацию пользователя"

### Комментарии в коде
```typescript
// Получаем список клиентов из базы данных
const clients = await db.select().from(clientsTable);

// Фильтруем только активных
const activeClients = clients.filter(c => c.isActive);
```

✅ "Добавлена фильтрация клиентов"
