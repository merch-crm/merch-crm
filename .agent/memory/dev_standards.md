# Стандарты разработки

- **Язык планов**: Все планы внедрения (implementation_plan.md) и списки задач (task.md) должны быть строго на **русском языке**.
- **Уведомления**: Использовать кастомный `toast` вместо `alert()`.
- **Подтверждения**: Использовать компонент `ConfirmDialog` вместо `confirm()`.
- **Роли**: Использовать русские названия ролей из БД ("Администратор", "Руководство", "Склад", "Отдел продаж").
