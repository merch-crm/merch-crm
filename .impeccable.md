# Impeccable Design Context: MerchCRM

This file serves as the design context for the `impeccable` and `frontend-design` skills.

## Core Audience & Context
- **Target Audience**: Back-office staff (Speed, Density), Workshop workers (Tactility, High-contrast, Visual status), Designers (Precision, Aesthetics).
- **Use Cases**: Managing order lifecycle, Production tracking, Inventory management, Financial reporting, Gamification.

## Brand Personality (The Aesthetic Direction)
- **Tone**: Professional, Precise, Reliable, Transparent.
- **Direction**: "Modern Industrial Craft".
- **Keywords**: Instrumental, Machine-precision, Refined craft, Clean-data.

## Visual Guardrails
- **Color Space**: OKLCH for smooth gradients and accessible palettes.
- **Typography**: Modular type scale with fluid clamp sizing. Avoid system defaults.
- **Motion**: Purposeful staggered reveals, ease-out-quart for deceleration.
- **Interactive**: Full cursor-pointer coverage, optimistic UI updates, clear visual focus.

## Mandatory Patterns
- Use centralized `UX_STANDARDS.md` in `.agent/` for technical specs (4px grid, component mappings).
- Maintain "Industrial Compactness" (avoid large gaps, prioritize information density).
- No standard "AI slop" (cyan/purple gradients, generic glassmorphism without purpose).
